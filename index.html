Carpeta plana de razonamiento

Convierte ruido en una tarea ejecutable.

PRP Lite usa Markdown para ordenar contexto, cerrar decisiones y entregar prompts XML compactos a otro agente. Menos ambigüedad, menos contexto perdido, más ejecución.

Ver flujo Explorar archivos
Flujo central
1. Idea / issue / bug
2. Investigación mínima
3. PRP Lite
4. Decisión y plan corto
5. Prompt XML final
6. Agente ejecutor + validación

Qué resuelve

Tareas con objetivo borroso, alcance inflado o validación ausente. PRP Lite fuerza la conversación correcta antes de escribir código.

Qué contiene

AGENTS.md, plantillas, checklists y un ejemplo. Todo pensado para subirlo a ChatGPT o a otro agente.

Qué produce

Decisiones cerradas, contexto mínimo, alcance permitido, zonas prohibidas y validación concreta.

Uso recomendado
  • Sube los archivos Markdown a tu carpeta de ChatGPT.
  • Usa AGENTS.md o 00_INSTRUCTIONS.md como instrucciones del proyecto.
  • Pega la issue, bug o feature con el contexto mínimo.
  • Pide primero el PRP Lite y no el prompt final hasta cerrar decisiones.

Hecho para equipos que quieren pensar mejor antes de delegar ejecución.
